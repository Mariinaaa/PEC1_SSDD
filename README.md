# PEC1_SSDD
Este es la prueba de evaluación contunia 1 de sistemas distribuidos en la que hay 4 ejercicios:
- El primero: simula la actualización de una variable compartida llamada counter a través de varios hilos.
- El segundo: modelo básico del protocolo de gossip (propagación de rumores) en Python.
- El tercero: reloj lógico verctorial
- El cuarto: un modelo de mensajería Emisor-Suscriptor utilizando unaestructura basada en AMQP, donde los productores publican mensajes sobre diferentes temas llamados topics (como sports, news, weather), y los consumidores están suscritos a esos topics para recibir los mensajes.
